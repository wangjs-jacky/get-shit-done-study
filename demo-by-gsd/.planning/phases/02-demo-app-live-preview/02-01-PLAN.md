  <action>
Create the PomodoroTimer React component with the following features:

1. **Timer State Management**:
   - Use useState for: time (seconds), isRunning, mode ('work' | 'break')
   - Work duration: 25 minutes (1500 seconds)
   - Break duration: 5 minutes (300 seconds)

2. **Timer Display**:
   - Format time as MM:SS (e.g., "25:00", "04:59")
   - Use large monospace font for readability
   - Apply CSS variable colors: var(--color-text) for numbers

3. **Control Buttons**:
   - Start/Pause toggle button (single button, text changes)
   - Reset button (returns to current mode's initial time)
   - Style buttons using var(--color-primary) for active states
   - Add hover/focus states for accessibility

4. **Mode Switching**:
   - Two buttons: "Work" and "Break"
   - Clicking a mode button resets timer to that mode's duration
   - Highlight active mode with var(--color-primary) border/background

5. **Circular Progress Ring**:
   - SVG circle with strokeDashoffset based on remaining time
   - Circumference calculation: 2 * PI * radius
   - Progress = (time / initialTime) * circumference
   - Use var(--color-primary) for progress stroke

6. **useEffect for Countdown**:
   - Use setInterval when isRunning is true
   - Clear interval when isRunning is false or component unmounts
   - Update time every second

7. **Styling**:
   - Use Tailwind CSS classes
   - Apply CSS variables for colors (enables style switching)
   - Dark theme by default (Terminal Noir)
   - Responsive: center content, max-width container

Component signature:
