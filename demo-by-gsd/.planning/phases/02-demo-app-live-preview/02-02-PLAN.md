Create the PreviewPane React component that displays the Pomodoro timer and current style information:

1. **Props Interface**:
```typescript
interface PreviewPaneProps {
  styles: Style[];
  selectedStyleId: string;
  onStyleSelect: (styleId: string) => void;
  className?: string;
}
```

2. **Component Structure**:
   - Main container with dark background (var(--color-bg))
   - PomodoroTimer centered in the preview area
   - Footer showing current style name and description
   - "Copy Prompt" button (functionality in Phase 3, just add button now)

3. **CSS Variable Injection** (CRITICAL for style switching):
   - Use useEffect to inject CSS variables when selectedStyleId changes
   - Find selected style from styles array
   - Apply variables to document.documentElement.style:
   ```typescript
   useEffect(() => {
     const selected = styles.find(s => s.id === selectedStyleId);
     if (selected) {
       Object.entries(selected.cssVariables).forEach(([key, value]) => {
         document.documentElement.style.setProperty(key, value);
       });
     }
   }, [selectedStyleId, styles]);
   ```

4. **Styling**:
   - Use Tailwind CSS with CSS variable colors
   - Glass effect background: var(--glass-bg), var(--glass-border)
   - Rounded corners, subtle border
   - Flexbox layout: timer in center, info at bottom

5. **Copy Prompt Button**:
   - Button with clipboard icon
   - Position in footer area
   - Use var(--color-primary) for button accent
   - onClick handler can be placeholder (Phase 3 implements clipboard)

Do NOT add tests in this task - only the component implementation.
